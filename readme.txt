User Guide
=================

Note: Haven't installed the extension? Read the readme.txt for guide!

Configuring Your Extension

1. Please go to system -> configuration -> Store Locator -> General in order to configure your extension.

2. You must save the configuration at least one time
	->Enter Default Latitude
	->Enter Default Longitude
	->Enable Store Locator
	->Add Storelocator to top menu(If yes then extension link will shown in top links http://awesomescreenshot.com/08525slqee)

3. You must delete all browser cookies cache after installation

4. You must delete magento cache after installation(var/cache) or system -> Cache managemet

Once you save the configuration then please add your store and it's location from
Storelocator -> Manage storelocator



Thank you
